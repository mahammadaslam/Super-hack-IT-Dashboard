## 1. Problem Statement
IT Teams struggle with missed updates, too many alerts, and failed backups - causing downtime and security risks.

## 2. Our Solution
The **Smart IT Ops Dashboard** is a lightweight, all-in-one platform that solves these three problems. It provides a "single-pane-of-glass" for IT admins to manage patches, filter critical alerts, and automate backups.

## 4. Key Features
* **Alert Filtering:** Instantly cuts 90% of non-critical alert "noise."
* **Task Execution:** One-click tasks (patch, backup) with instant feedback.
* **AI Co-Pilot Speed:** AI helper generates reports, scripts, and analysis in ~2 seconds.
* **Dashboard Speed:** Zero load time between pages (it's a Single Page App).

## 5. Technologies Used
* **Frontend:** HTML, CSS, Bootstrap, JavaScript
* **Backend:** Python (Flask/Django)
* **Database:** SQLite
* **Automation:** Cron jobs / Task Scheduler
* **Notifications:** SMTP, Slack API
